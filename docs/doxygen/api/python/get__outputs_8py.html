get_outputs()¶
module get_outputs¶

Get all output tensor buffers of runner.

Sample code:

output_tensor_buffers = runner.get_outputs()
output_element_num = tuple(output_tensor_buffers[0].get_tensor().get_element_num())
Returns:

List[vart.TensorBuffer]. All output tensors. A vector of raw pointer to the output tensor.
